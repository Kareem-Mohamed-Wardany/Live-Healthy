@startuml
title Create Prescriptions
actor       Doctor       as User
boundary    DoctorPage  as Page
control     SystemHandler     as SystemHandler
entity      Doctor as Doctor



User -> Page : Prescription icon
Page -> SystemHandler: Load Create Prescriptions Frame
SystemHandler-> Doctor:HandlePrescription
Doctor-> Doctor:PrescriptionGenerated()
Doctor--> SystemHandler:Result
alt Not Empty Result
    SystemHandler --> Page : Show Created Prescription
else
    SystemHandler --> Page : Show Prescription Fill Up Window
    User -> Page : Add icon
    Page -> SystemHandler: Add Medicine Entry
    User -> Page : Save
    Page -> SystemHandler: Save Prescription
    SystemHandler-> Doctor:MakePrescription()
    Doctor--> SystemHandler:Confirm
    Doctor--> SystemHandler:Show info Message "Prescription Created"
end
@enduml
